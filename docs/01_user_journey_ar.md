# رحلة المستخدم

## الهدف

تمكين الشركة من تشغيل التسويق الرقمي باستخدام الذكاء الاصطناعي مع وجود
مراجعة واعتماد بشري قبل التنفيذ.

## المرحلة 1 -- التسجيل

-   إنشاء حساب الشركة.
-   اختيار القطاع.
-   تحديد البلد واللغة.
-   تحديد الهدف: ليدز / مكالمات / مبيعات / زيارات.

## المرحلة 2 -- ربط الحسابات

-   الموقع الإلكتروني.
-   Google Analytics.
-   Google Search Console.
-   حسابات الإعلانات.
-   صفحات السوشيال.

## المرحلة 3 -- التحليل التلقائي

-   قراءة الموقع.
-   تحليل الأداء.
-   اكتشاف مشاكل SEO.
-   تحليل المنافسين.
-   دراسة الكلمات المفتاحية.

## المرحلة 4 -- إنشاء خطة النمو

-   خطة محتوى شهرية.
-   خطة إعلانات.
-   خطة تحسين SEO.
-   مقترحات تصميم.

## المرحلة 5 -- المراجعة والاعتماد

✔ موافقة\
✔ تعديل\
✔ رفض

## المرحلة 6 -- التنفيذ

بعد الموافقة يتم إنشاء ونشر الحملات والمحتوى.

## المرحلة 7 -- استقبال الليدز

تجميع وتوزيع على موظفي المبيعات.

## المرحلة 8 -- التقارير

تكلفة الليد، أفضل القنوات، توصيات التحسين.

## Auto Mode

تنفيذ تلقائي ضمن حدود يتم تحديدها.
