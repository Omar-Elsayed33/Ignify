### DESIGN SPECIFICATIONS

Layout:
- Max width: 1280px centered
- Section padding: 80px vertical (mobile: 48px)
- Grid: 12-column, gap 24px
- Border radius: 12px cards, 8px buttons, 24px hero elements
- Box shadows: 0 4px 20px rgba(0,0,0,0.08)

Animations (subtle):
- Cards: hover translateY(-4px) + shadow increase
- Buttons: hover scale(1.02) + brightness increase
- Section entry: fade-in + translateY(20px) on scroll (Intersection Observer)
- Numbers: count-up animation on first view

Mobile responsive:
- Breakpoints: 640px (sm), 768px (md), 1024px (lg), 1280px (xl)
- Navigation collapses to hamburger at <1024px
- Cards stack to 1 column at <640px
- Hero text size reduces proportionally
- Touch targets minimum 44px

Accessibility:
- All images have Arabic alt text
- Form labels properly associated
- Color contrast WCAG AA compliant
- Focus states visible
